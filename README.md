## To Clone
If you wish to clone this project, make sure to clone all the submodules too:
```
git clone --recursive https://github.com/SzyJ/midori_engine
```
or
```
git clone https://github.com/SzyJ/midori_engine
cd midori_engine
git submodule init
git submodule update
```
